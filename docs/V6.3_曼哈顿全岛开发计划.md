# V6.3 曼哈顿全岛开发计划

## 版本定位

V6.3 不再延续五区纽约隐喻，也不再继续修补 V6.2 的地理底盘。

主方案固定为：

> 曼哈顿全岛 + Liberty Island + Governors Island + Grey Island

目标不是做一张知识地图，而是做一座可以承载张宇《基础30讲》第15讲《微分方程》的真实城市学习模型。

## 已确定原则

1. 主城是曼哈顿狭长岛。
2. Liberty Island 独立，承载概念入口。
3. Governors Island 独立，承载风险训练场。
4. Central Park 是唯一允许的大空地，承载几何应用。
5. Battery Park 可以作为概念登陆口。
6. East River Piers 承载物理应用。
7. 普通曼哈顿街区不允许出现无意义大空地。
8. 章节边界采用弱提示，不使用大面积彩色沙盘。
9. 建筑必须分为城市肌理建筑和知识地标建筑两层。
10. 临时 gh-pages 预览只用于快速验证，长期必须回到模块化工程。

## 章节到曼哈顿空间映射

| 章节模块 | 城市空间 | 城市语义 |
| --- | --- | --- |
| 微分方程概念 | Liberty Island + Battery Park | 入城、概念火炬、通解/特解登陆口 |
| 一阶微分方程 | SoHo / Greenwich Village | 中低层方法街区 |
| 二阶可降阶 | Chelsea / Hudson Yards / High Line | 工业加工、变量角色转换、阶数下降 |
| 高阶线性 | Midtown / Times Square | 最高、最密、最核心的天际线 |
| 几何应用 | Central Park | 切线、轨迹、自行车后轮运动 |
| 物理应用 | East River Piers | 跑道、码头、实验塔、现实建模 |
| 风险点 | Governors Island | 独立风险训练场 |
| 差分方程 | Grey Island | 远景选学区 |

## V6.3.1：地理底盘

验收标准：

- 曼哈顿第一眼是狭长岛，不是知识沙盘。
- Liberty Island 不连着曼哈顿。
- Governors Island 不连着曼哈顿。
- Central Park 是唯一大空地。
- 普通街区由建筑肌理填满。
- 大章节分区通过地理位置、道路、岸线、门牌、小地图表达。

## V6.3.3：学习可读性

- 远景只显示大区名。
- 中景显示核心地标名。
- 近景取消漂浮标签，依赖门牌和点击面板。
- 夜景显示章节主线、换元线、线性化线、风险线。
- 小地图固定显示章节位置，但不能遮挡主视图。
