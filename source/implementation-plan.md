# V6 后续工程化实现计划

## 目标

把当前仓库版 V6 从静态城市规划页升级为真正的 WebGL / Three.js / 原生 WebGL 可探索城市。

## 开发优先级

### 第一阶段：城市形态

- 微分河水体。
- 曼哈顿核心岛。
- 皇后一阶城区。
- 布鲁克林工业区。
- 应用海湾。
- 风险训练场。
- 灰岛。

### 第二阶段：地标建筑

- 特征方程中央塔。
- 重根摩天楼。
- 复根旋转塔。
- 非齐次特解工厂。
- 一阶线性馆。
- 伯努利转换门。
- 可降阶电梯。
- 欧拉跨江大桥。
- 自行车轨迹馆。
- 飞机减速跑道。
- 牛顿冷却实验塔。

### 第三阶段：知识基础设施

- 张宇主线大道。
- 换元桥。
- 线性化地铁线。
- 应用建模线路。
- 风险提醒线。

### 第四阶段：交互

- 斜俯视旋转镜头。
- 区域飞行。
- 点击城区打开知识面板。
- 标签远近显隐。
- 白天模式与知识夜景模式。

### 第五阶段：深入学习

城市建模稳定后，再加入建筑内部学习：

- 一阶线性馆：积分因子机器。
- 伯努利馆：非线性转线性门。
- 可降阶馆：二阶下降电梯。
- 高阶线性塔：特征方程转换室。
- 应用海湾：现实建模场景。

## 不允许回退的错误方向

- 不允许再做二维节点连线网。
- 不允许把建筑做成无意义盒子。
- 不允许漂浮标签污染主视图。
- 不允许为了炫技牺牲学习清晰度。
- 不允许内部学习内容优先于城市建模本身。
